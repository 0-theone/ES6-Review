In the Browser (i.e. as part of websites)

JavaScript code can be included in any website, allowing it to execute within the visitor's browser. This means that the code runs directly on the visitor's machine.

JavaScript-Code kann in jede Webseite eingebunden werden und wird dann im Browser des Besuchers ausgeführt, also auf dessen Gerät.

On any computer (e.g, server-side code)

Thanks to Node.js or Deno, JavaScript code can be executed outside of the browser as well. In this case, the code runs directly on the machine.

Dank Node.js oder Deno kann JavaScript-Code auch außerhalb des Browsers ausgeführt werden. In diesem Fall erfolgt die Ausführung direkt auf dem Rechner.

On mobile Devices

By utilizing additional technologies such as Capacitor or React Native, it is possible to develop mobile applications using JavaScript. In this scenario, the code is executed directly on the mobile device.

Dank zusätzlicher Technologien wie Capacitor oder React Native können mobile Apps mit JavaScript entwickelt werden. In diesem Fall wird der Code direkt auf dem mobilen Gerät ausgeführt.
